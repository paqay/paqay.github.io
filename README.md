## 🌐 Overview
Welcome to the repository for **paqay.cc**, the official landing page for paqay – an electronic music producer and rapper from Vienna. 

The site features a dynamic, immersive experience with:
-   🎨 **Glassmorphic UI**: Modern aesthetic with blurred backgrounds and difference blend modes.
-   🎬 **Dynamic Backgrounds**: Randomly loaded video footage with a custom lo-fi lowpass audio filter.
-   📱 **Fully Responsive**: Optimized for all devices, from desktop to mobile.
-   📧 **Quick Contact**: Integrated direct mail connection.
